### 🚀 Getting Started

```bash
# 1. Clone the project
git clone https://github.com/Shantanu7854/JSON-Schema-Builder
cd json-schema-builder

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Now visit [http://localhost:5173](http://localhost:5173) in your browser.
